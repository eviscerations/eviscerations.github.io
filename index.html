whisper-windows-mcp
Native Windows MCP server for local AI transcription
v2.3.0 stable

A native MCP (Model Context Protocol) server for Windows built on whisper.cpp. Transcribes audio and video files locally in Claude Desktop with Vulkan GPU acceleration, multilingual support, batch processing, and a full privacy architecture for regulated content. No cloud. No CUDA requirement. No macOS assumptions.

Runtime whisper.cpp + Vulkan
GPU AMD · NVIDIA · Intel
Platform Windows 10 / 11
Languages 99+ via large-v3
Privacy mode metadata-only output
License MIT · Commercial

amd · rocm · comfyui · gfx1031
comfyui-rinsei
Native ROCm acceleration for unsupported RDNA2 on Windows
in development

Native AMD ROCm GPU acceleration for ComfyUI on gfx1031 (RX 6700 XT) and the full gfx103x series on Windows. AMD's official portable crashes on generate for this hardware. This project makes it work — image generation, video generation (Wan 2.2, HunyuanVideo), and the full ComfyUI ecosystem running on hardware AMD refused to support.

Target GPU gfx1031 · RX 6700 XT
Series gfx1030 · 1031 · 1032
Platform Windows 10 / 11
Backend TheRock nightly wheels
Video models Wan 2.2 · HunyuanVideo
Image models Flux · Z-Image · SDXL

houdini · lidar · mcp · paleoglaciology
houdini-lidar-mcp
Cinema-quality glacial reconstruction from real 1-meter LiDAR data
active dev

MCP server for Houdini integrating real 1-meter USGS LiDAR data for cinema-quality geological VFX. Built on seven years of independent research into the Younger Dryas Impact Hypothesis. Reconstructs the Cordilleran Ice Sheet and Glacial Lake Missoula from actual terrain data — KMZ glacier boundary files, point cloud cleanup pipelines, Redshift lookdev, and iRender farm output.

Data source 1m USGS LiDAR · Montana/Idaho
Pipeline LiDAR → Houdini → Redshift
Subject Younger Dryas Impact · YDIH
Render iRender farm · Houdini 21
Output .obj · .fbx · .abc · Disguise d3
Research 300+ peer-reviewed papers
