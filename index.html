// How it works

THREE STEPS.
THAT'S IT.

01
CONNECT KICK

Click the button, authorize with your Kick account via the official OAuth 2.1 flow. No passwords shared, no permissions you don't approve.

OAUTH 2.1 + PKCE
02
PASTE WEBHOOK

Copy a Discord webhook URL from any channel you own and paste it in. Takes 30 seconds. Discord server → channel settings → Integrations → Webhooks.

DISCORD WEBHOOK
03
GO LIVE.

That's it. Every time you go live on Kick, your Discord community gets a notification automatically. No button to press. No forgetting.

FULLY AUTOMATIC
kick-notifier — event received: livestream.status.updated
✓ signature verified (RSA PKCS1v15)
✓ broadcaster: your_channel — is_live: true
✓ discord webhook found
→ Discord notification sent: "Your Stream Title"
